# seamless_voice_translation_with_llm_integration
Developed a real-time multilingual voice translation system that captures live speech, transcribes it using Google Speech API,
and translates the content into the target language using a large language model (LLM). Deployed a user-friendly interface using
Streamlit for seamless interaction across devices. Enhanced translation fluency by leveraging context-aware prompts for the LLM
to produce natural-sounding output.
